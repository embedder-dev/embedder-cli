# Embedder CLI

AI-powered CLI tool for embedded systems development.

## Installation

```bash
curl -fsSL https://raw.githubusercontent.com/embedder-dev/embedder-cli/main/install.sh | bash
```

For Windows (PowerShell):
```powershell
irm https://raw.githubusercontent.com/embedder-dev/embedder-cli/main/install.ps1 | iex
```
